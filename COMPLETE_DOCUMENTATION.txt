PROJECT OVERVIEW
================================================================================
Glassy Tableau is a browser extension that transforms your new tab page into a
beautiful, glassmorphic interface with unlimited customizable tiles, folders,
notes, to-do lists, and more. Inspired by Yandex Browser's elegant design.

Version: 1.0.0
Manifest Version: 3
Compatible Browsers: Microsoft Edge, Google Chrome, Chromium-based browsers


CORE FEATURES
================================================================================

1. UNLIMITED TILES

TECHNICAL ARCHITECTURE
================================================================================

STORAGE SYSTEM:
--------------
1. Chrome Storage API (chrome.storage.local)
   - Tiles data
   - Settings
   - Quotes deck and index
   - Notes
   - To-do lists

2. IndexedDB (for large files)
   - Wallpapers (images and videos)
   - Custom tile icons
   - Wallpaper folder references

STORAGE KEYS:
------------
- tiles: Array of tile objects
- settings: Settings object
- quotesDeck: Shuffled quotes array
- quotesIndex: Current quote index
- userNotes: Array of note objects
- userTodos: Array of to-do list objects

DATA STRUCTURES:
---------------

Tile Object:
{
  id: string,              // Unique identifier (timestamp)
  name: string,            // Display name
  url: string,             // Website URL
  icon: string,            // Icon URL or data URL
  type: 'link' | 'folder', // Tile type
  color: string,           // Background color (optional)
  items: Array,            // Folder items (if type is 'folder')
  focusEnabled: boolean    // Focus mode flag (optional)
}

Settings Object:
{
  tileSize: 'small' | 'medium' | 'large',
  showQuotes: boolean,
  quotePosition: 'both' | 'left' | 'right',
  quoteTextColor: string,
  timeFont: string,
  timeColor: string,
  dateColor: string,
  searchEngine: 'google' | 'bing' | 'duckduckgo',
  timeFormat: '12h' | '24h',
  tileTheme: string,
  applyThemeToSearchbar: boolean,
  focusModeActive: boolean,
  darkModeEnabled: boolean,
  keyboardShortcuts: object
}

Note Object:
{
  id: string,              // Unique identifier
  title: string,           // Note title
  content: string,         // Note content
  source: string,          // Source URL (optional)
  createdAt: number,       // Timestamp
  updatedAt: number        // Timestamp
}

To-Do List Object:
{
  id: string,              // Unique identifier
  title: string,           // List title
  items: Array,            // Array of to-do items
  createdAt: number,       // Timestamp
  updatedAt: number        // Timestamp
}

To-Do Item Object:
{
  id: string,              // Unique identifier
  text: string,            // Task text
  completed: boolean,      // Completion status
  createdAt: number        // Timestamp
}

INSTALLATION GUIDE
================================================================================

STEP 1: DOWNLOAD FILES
----------------------
Download all extension files to a folder on your computer.
Keep the folder structure intact:
  - All .js, .html, .css, .json files in root
  - icons/ folder with all icon files

STEP 2: LOAD IN BROWSER
-----------------------
1. Open Microsoft Edge or Chrome
2. Navigate to edge://extensions/ or chrome://extensions/
3. Enable "Developer mode" (toggle in bottom-left or top-right)
4. Click "Load unpacked"
5. Select the folder containing the extension files
6. Extension should now appear in your extensions list

STEP 3: VERIFY INSTALLATION
---------------------------
1. Open a new tab - you should see the Glassy Tableau interface
2. Click the extension icon in toolbar to test Quick Add
3. Check settings by clicking the settings button (‚öôÔ∏è)

STEP 4: OPTIONAL CONFIGURATION
------------------------------
1. Upload a custom wallpaper
2. Adjust tile size
3. Configure search engine
4. Enable/disable quotes
5. Set up keyboard shortcuts
6. Enable dark mode

TROUBLESHOOTING
================================================================================

EXTENSION DOESN'T LOAD:
-----------------------
- Ensure all files are in the same folder
- Check that Developer mode is enabled
- Try removing and re-adding the extension
- Check browser console for errors (F12)

NEW TAB PAGE DOESN'T SHOW:
-------------------------
- Restart browser after installing
- Check that extension is enabled in extensions page
- Try opening new tab in Incognito mode
- Ensure no other new tab extensions are conflicting

TILES DON'T SAVE:
----------------
- Check browser console for errors (F12)
- Ensure extension has storage permissions
- Try clearing browser cache
- Check available storage space

DRAG AND DROP NOT WORKING:
-------------------------
- Ensure you're clicking and holding on the tile
- Drag over another tile until you see highlight
- Release to create folder or reorder
- Check that tiles are not in edit mode

WALLPAPER NOT UPLOADING:
-----------------------
- Check file size (images: 5MB max, videos: 3MB max)
- Ensure file format is supported
- Try a different image/video
- Check browser console for errors

DARK MODE NOT WORKING:
---------------------
- Ensure dark mode is enabled in settings
- Check that content script has permissions
- Some sites may already have dark themes
- Try refreshing the page

QUOTES NOT SHOWING:
------------------
- Check that "Show quotes" is enabled in settings
- Ensure quotes.json file is present
- Check quote position setting
- Try refreshing the page

KEYBOARD SHORTCUTS NOT WORKING:
------------------------------
- Ensure you're not typing in an input field
- Check that shortcuts don't conflict with browser shortcuts
- Try clicking on the page background first
- Check keyboard shortcuts help (? or H)
